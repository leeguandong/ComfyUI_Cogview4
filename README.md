# ComfyUI nodes to use Cogview4

https://github.com/THUDM/CogView4


## Install

- 推荐使用管理器 ComfyUI Manager 安装（On the Way）

- 手动安装：
    1. `cd custom_nodes`
    2. `git clone https://github.com/leeguandong/ComfyUI_Cogview4.git`
    3. 重启 ComfyUI
